Apple Calendar (Mac/iPhone)

  1. Copy the URL above
  2. Open Calendar app
  3. File → New Calendar Subscription (Mac) or
    Settings → Calendar → Accounts → Add Account → Other → Add Subscribed Calendar (iPhone)
  4. Paste the URL and click Subscribe

Google Calendar

  1. Copy the URL above
  2. Open Google Calendar
  3. Click the + next to "Other calendars"
  4. Select "From URL"
  5. Paste the URL and click "Add calendar"
